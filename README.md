## 🚀 使用方法 (How to Use)

### 🇨🇳 中文指南

- **Step 1 — 上傳圖片**：將圖片拖放至頁面中央，或點擊上傳區域選擇本地檔案（支援 JPG / PNG）。
- **Step 2 — 選擇處理模式**：在右側控制面板頂部選擇風格模式，共 12 種可選，包括半調、ASCII、故障、版畫、復古等。
- **Step 3 — 調整參數**（可選）：根據不同模式調整強度、顏色、細節等參數。
- **Step 4 — 預覽與下載**：即時預覽處理結果，滿意後點擊下載按鈕儲存圖片。

### 🇬🇧 English Guide (Optional)

- **Step 1** — Upload Image: Drag and drop image to the center area, or click to select local file (JPG/PNG supported).
- **Step 2** — Select Style: Choose from 12 different styles in the control panel (Halftone, ASCII, Glitch, Woodcut, Vintage, etc.).
- **Step 3** — Adjust Parameters (Optional).
- **Step 4** — Preview & Download.
